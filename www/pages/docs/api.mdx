# API Reference

## `HybricEditor` Props

| Prop | Type | Description |
| --- | --- | --- |
| `content` | `string` | Initial Markdown content. |
| `onChange` | `(editor) => void` | Returns the editor instance on updates. |
| `onExtract` | `(data) => void` | Callback when user clicks "Extract to Card". |
| `className` | `string` | Custom Tailwind classes for the editor wrapper. |

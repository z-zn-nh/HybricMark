## <I18nText en="Adding your own extension" zh="添加自定义扩展" />

```tsx
import CharacterCount from '@tiptap/extension-character-count'

<HybricEditor
  content={doc}
  extensions={[
    CharacterCount.configure({ limit: 20000 }),
  ]}
/>
```

> <I18nText en="Recommendation: Keep business-specific features (mentions, embeds, workflow states) in custom extensions, while leaving core writing UX to HybricMark defaults." zh="建议：业务特性（@提及、嵌入、流程状态）放在自定义扩展层，核心写作体验沿用 HybricMark 默认能力。" />

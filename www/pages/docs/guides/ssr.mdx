# <I18nText en="Next.js / SSR" zh="Next.js / SSR" />

<I18nText en="Editor surfaces must run on the client." zh="编辑器渲染面必须在客户端运行。" />

## <I18nText en="Recommended pattern" zh="推荐模式" />

```tsx
'use client'

import dynamic from 'next/dynamic'

const HybricEditor = dynamic(
  () => import('hybricmark').then((mod) => mod.HybricEditor),
  { ssr: false },
)
```

## <I18nText en="IME and cursor stability" zh="IME 与光标稳定性" />

<I18nText en="Do not force-update `content` from parent state on every keypress." zh="不要在每次按键时从父状态强制回写 `content`。" />

<I18nText en="Use:" zh="正确做法：" />

- <I18nText en="`content` as initial value." zh="`content` 仅作为初始值。" />
- <I18nText en="`onChange` for immediate UI." zh="`onChange` 负责即时 UI 联动。" />
- <I18nText en="`onDebouncedUpdate` for persistence." zh="`onDebouncedUpdate` 负责持久化写入。" />

# <I18nText en="Getting Started" zh="快速开始" />

## <I18nText en="Prerequisites" zh="前置条件" />

- <I18nText en="React 18+" zh="React 18 及以上" />
- <I18nText en="Tiptap peer dependencies available in your app" zh="应用中已安装 Tiptap 对等依赖" />
- <I18nText en="Client-rendered surface for the editor view (especially in Next.js)" zh="编辑器渲染区域需在客户端运行（尤其是 Next.js）" />

## <I18nText en="Install" zh="安装" />

```bash
npm install hybricmark @tiptap/react @tiptap/starter-kit
```

## <I18nText en="Minimal integration" zh="最小集成" />

```tsx
import { HybricEditor } from 'hybricmark'

export default function EditorPage() {
  return (
    <HybricEditor
      content="# Hello\n\nStart typing..."
      onChange={(editor) => {
        console.log(editor.getJSON())
      }}
    />
  )
}
```

## <I18nText en="Production baseline" zh="生产环境基线配置" />

```tsx
<HybricEditor
  content={initialDoc}
  placeholder="Type '/' for commands..."
  debounceMs={800}
  onDebouncedUpdate={({ editor }) => {
    void saveToAPI(editor.getJSON())
  }}
/>
```

## <I18nText en="Styling" zh="样式接入" />

<I18nText en="Import editor styles once in your app entry:" zh="在应用入口只需引入一次编辑器样式：" />

```ts
import 'hybricmark/style.css'
import 'katex/dist/katex.min.css'
```

<I18nText en="Then attach your own wrapper class if needed:" zh="然后按需给编辑器挂载业务外层类名：" />

```tsx
<HybricEditor className="my-editor-shell" content={content} />
```

## <I18nText en="Next.js (client-only)" zh="Next.js（仅客户端渲染）" />

```tsx
'use client'

import dynamic from 'next/dynamic'

const HybricEditor = dynamic(
  () => import('hybricmark').then((mod) => mod.HybricEditor),
  { ssr: false },
)
```

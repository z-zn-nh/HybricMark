## <I18nText en="Typical integration" zh="典型接入方式" />

```tsx
<HybricEditor
  content={doc}
  className="my-editor-shell"
  editorProps={{
    attributes: {
      class: 'prose max-w-none leading-7',
    },
  }}
/>
```

```css
.my-editor-shell .hm-editor-content h1 {
  font-size: 2rem;
  border-bottom: 1px solid #e5e7eb;
}

.my-editor-shell .hm-editor-content code {
  background: #f3f4f6;
  border-radius: 4px;
  padding: 0.15rem 0.35rem;
}
```

## <I18nText en="Best practices" zh="最佳实践" />

- <I18nText en="Scope typography rules under a local wrapper class." zh="所有排版规则都挂在本地 wrapper 下。" />
- <I18nText en="Keep command/state logic in TS, keep visual decisions in CSS/Tailwind." zh="命令与状态逻辑放在 TS，视觉决策放在 CSS/Tailwind。" />
- <I18nText en="Always verify dark mode contrast for links, selection, table, and code." zh="务必验证暗色模式下链接、选中态、表格和代码块对比度。" />
